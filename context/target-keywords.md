# Keywords Objetivo por Cluster Temático — Ferrolan

Organización de keywords objetivo por cluster temático para planificación de contenido SEO.

## Estructura de Clusters

Cada cluster tiene:
- **Keyword pilar**: Principal, alto volumen (competitivo)
- **Keywords cluster**: 5-10 keywords relacionadas (subtemas)
- **Keywords long-tail**: Frases específicas de menor volumen
- **Intención de búsqueda**: Qué busca el usuario
