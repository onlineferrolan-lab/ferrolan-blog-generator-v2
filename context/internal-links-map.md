## Referencia Rápida por Tema

**Cuando escribas sobre baños**, enlazar a:
- `/banos` — Categoría principal de baños
- `/banos/lavabos` — Si se mencionan lavabos específicamente
- `/azulejos` — Si se habla de revestimiento de paredes

**Cuando escribas sobre cocinas**, enlazar a:
- `/cocinas` — Categoría principal
- `/azulejos` — Si se habla de salpicadero o revestimiento
- `/construccion` — Si se habla de reforma estructural

**Cuando escribas sobre suelos**, enlazar a:
- `/parquet` — Para suelos de madera, laminados, vinílicos
- `/azulejos` — Para suelos cerámicos, gres, porcelánico
- `/construccion` — Si se habla de preparación del soporte

**Cuando escribas sobre exterior/jardín**, enlazar a:
- `/jardineria` — Herramientas, plantas, riego
- `/azulejos` — Pavimento exterior, gres antideslizante
- `/construccion` — Materiales para muros, pérgolas, cimentación

**Cuando escribas sobre bricolaje/DIY**, enlazar a:
- `/ferreteria` — Herramientas y accesorios
- `/construccion` — Materiales básicos
- La categoría relevante según el proyecto

## Buenas Prácticas de Enlazado

1. **Enlazar solo cuando sea relevante**: No forzar enlaces a categorías que no tienen relación
2. **Variar el anchor text**: Usar frases diferentes para la misma URL en distintos artículos
3. **2-4 enlaces internos por artículo**: Ni más ni menos
4. **Deep linking**: Enlazar a páginas específicas, no siempre a la home
5. **Distribuir**: Repartir los enlaces por el cuerpo del artículo, no concentrar al final
6. **Actualizar este mapa**: Añadir nuevas páginas cuando se creen
